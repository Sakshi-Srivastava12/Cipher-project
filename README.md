# Cipher-project
This project demonstrates encryption and decryption techniques using classical ciphers in Python.
## 📁 Features

- Caesar Cipher
- Vigenère Cipher
- Substitution Cipher
- Brute-force decryption (for Caesar)
- Custom user input support

## 🛠️ Technologies Used

- Python 3
- File handling
- String manipulation
